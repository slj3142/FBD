# EaFort: Easy Fortran Handling Library



## Description

The **EaFort** is a collective library to provide an easy interface to develop the Fortran project.

## How to install

### Debian/Ubuntu

```bash
cmake -S. -Bbuild
cmake --build build
```

**Verified:**

- Debian 10 (buster), gfortran v8.3.0

### Windows

```powershell
cmake -S. -Bbuild
cmake --build build
```

**Verified:**

* OneAPI 2022 with intel fortran
